Por: Krusher, Laff, Anton Sapristi

¡Del deporte también se sale!

Faraónico megaproyecto balompédico producido por Charnego Translations Inc.
junto con el Comité para la Charnegación. Fork del YSoccer, clon del clásico
Sensible Soccer para Amiga. Siempre en construcción, pero de código libre, se
puede probar sin compromiso. No le devolvemos su dinero si no queda satisfecho
porque es gratis. De vez en cuando publicaramos (cuando nos pique) algún
compilado listo para ejecutar. Si te bajas la versión de desarrollo requiere
tener instalado Java 1.8 como mínimo. Debería funcionar en Wuindours, Leenucs
y MacacOS X. En realidad, en cualquier sitio que tenga instalado Java 8.

INSTRUCCIONES

Se proporcionan versiones para Windowrs, Leenucs y MacAcOs, listas para
viciarse. En otros sistemas operativos, usa Java 8 para ejecutar el .jar que 
hay en cualquiera de las versiones. ¡Búscate la vida, tío!

(En Windorws ejecuta el fichero LaffSoccer.cmd, en Leenucs el LaffSoccer.sh y
en Mak el LaffSoccer.command, no me seas brutísimo)

CONTROLES POR DEFESTO

Jugador 1:
Cursores - Movimiento
M - Botón 1 (aceptar, tiro, pase, etc.)
N - Botón 2 (sustituciones, cancelar)

Jugador 2:
WASD - Movimiento
V - Botón 1 (aceptar, tiro, pase, etc.)
B - Botón 2 (sustituciones, cancelar)

Cosas:

F1 - Mostrar ayuda
F2, F3 - Volumen de los ruíos
F4 - Apagar o encender comentarios
F5 - Apagar o encender cánticos
F6 - Cambiar entre ventana y pantalla completa
F7, F8 - Zoom parriba o pabajo
F9 - Repetisione artomáticas
F10 - Apagar o encender radar
P - Pausa
R - Repetisión
R - Cámara lenta
Boton 2 durante la repetisión - Cámara lenta de 4 velosidade
Espasio - Guardar acción para que salga en las repetisiones
H - Mejores jugadas (cuando termina el partío)
ESC - Te piras del partido
